How to Prepare to be a Computer Science Major

  • Don't Procrastinate - You need time to figure out errors in code
  • Learn Math - Math is important to write algorithms for computers
  • Don't be a loner - Working as a team prepares you for the real world
  • Learn to write - Documentation and Research papers are prevelant in this field. Learn to write.
  • Use resources - Google information if you are stuck on a problem
